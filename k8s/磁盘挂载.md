# 磁盘挂载

```bash
fdisk -l
fdisk  /dev/vdb

n->p->1->回车->回车->w

mkfs.xfs /dev/vdb -f
mkdir /data
mount /dev/vdb /data

vim /etc/fstab
/dev/vdb   /data     xfs       defaults  0 0
```

### 缩小分区

[Resize Partition and Filesystem with fdisk & resize2fs - GeekPeek.Net](https://geekpeek.net/resize-filesystem-fdisk-resize2fs/)

```bash
fuser -k -m /data
umount -f -R /data
e2fsck -f /dev/vdb1
resize2fs -p /dev/vdb1 1000G
fdisk /dev/vdb
p d p n +1000G p n p w
mount /dev/vdb1 /data
lsblk -f
```
